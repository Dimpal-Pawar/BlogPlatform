
# 📝 Blog Platform - Flask Web Application

A full-stack blogging platform developed using Python Flask, SQLite, HTML, and CSS.

## 🚀 Features

- User Registration & Login
- Secure Password Hashing
- Session Based Authentication
- Create, Edit & Delete Blogs
- Public Blog Viewing
- Like System
- Comment System
- User Profile Dashboard
- Blog Search Functionality
- Admin Dashboard

## 🛠️ Technologies Used

- Python
- Flask
- SQLite
- HTML5
- CSS3
- Jinja2 Templates
